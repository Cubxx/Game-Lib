# 游戏说明
## 玩家（自机）
红色 中心点为判定点   
最高生命值为5
最高开大数为5
## 敌机
黑色
不同关卡生命值不同，分别为10 15 50 300 400 500 
# 游戏事件
敌机生命值越少越透明  
得分每+180，开大数+1  
击破敌机：+20分，掉出4个生命值资源（每个+1/32生命值、+0.1分）  
开大：火力*4，无敌时间3s，消除敌机子弹  
被撞：生命值-1，无敌时间2s
